이 프로젝트 디렉토리는 에리엔진 전용 프로젝트입니다.

만약 node.js 개발에 익숙한 개발자라면, 이 디렉토리의 구조 또한 익숙할 것이라 생각합니다.
하지만 package.json의 내용물을 npm 명령줄로 직접 실행하지 마세요.

왜냐하면 실제 게임으로 빌드하는 과정에는 몇 가지 추가적인 작업이 필요하기 때문입니다.
이 추가작업은 에리엔진 내부에서 자동으로 처리하지만, npm으로는 처리되지 않습니다.

또한 dependencies, devDependencies에 직접 모듈을 추가하지 마세요.
package.json 파일은 유지보수를 위해 엔진을 킬 때 마다 덮어쓰기되고, 모듈을 새로 설치하기 때문입니다.

만약 추가해야하는 모듈이 있다면, package.json이 아니라, extend/package.json 파일에 기록해주세요.
이는 package.json 파일을 건드리지 않고, 모듈을 추가로 설치할 수 있도록 확장시켜줍니다.
